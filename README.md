# 疑似ブラウザアプリ

<h2>アプリ概要</h2>

　WebビューとDBを用いて、ブラウザが持つ機能(ブックマーク、履歴機能など)を疑似的に再現したアプリ。Googleをスタートページに  
 設定して検索を可能にし、閲覧途中でGoogleに戻りたい場合には、Googleに戻るためのボタンを用意。  
 　また、ブックマークには任意のコメントを追記することができる。
